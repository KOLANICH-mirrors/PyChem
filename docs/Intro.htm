Introduction

This package was designed for the analysis of 2D arrays of multivariate data.  Tools are provided for data pre-processing, categorical and quantitative modelling as well as feature selection or reduction.  Graphical reports can be saved or copied as metafiles, and in addition all results can be exported in ASCII text file format.

This help documentation describes how to use the software but is not intended as a multivariate analysis tutorial.  For more background on the subject of chemometrics the books cited below can provide a good starting point.

HEALTH WARNING

Although PyChem is intended to be intuitive and easy to use, it is also intended for those with an understanding of how to interpret the results of multivariate analysis.
