Spectral Pre-processing

A number of pre-processing algorithms for scaling, filtering, baseline correction and derivatisation are available.  The options available will be expanded in future releases of the software.

Applying Pre-processing Algorithms

Press the parameters button  to reveal the list of available pre-processing functions.

Double click on the function of choice to select it, then to compute the transformations select Processed spectra from the dropdown box and finally hit the  icon.  The functions are executed in the order they are listed in the Selected functions list box.  Functions can be reordered and removed or the selection box reset using the buttons beneath the list box.

NOTE:  The Run icon () must be pressed after selecting pre-processing steps in order to generate an array of processed spectral data.

Plotting Spectra

To plot spectra, simply select one of the All or Single options from the dropdown menu, then click the plot button.
