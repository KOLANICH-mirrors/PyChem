Spectral Pre-processing

A number of pre-processing algorithms for scaling, filtering, baseline correction and derivatisation are available.  The options available will be expanded in future releases of the software.

Applying Pre-processing Algorithms

Double click on the function of choice to select it, then to compute the transformations select the Processed radio button and finally hit the Plot button.  The functions are executed in the order they are listed in the Selected list box.  Functions can be reordered and removed or the selection box reset using the buttons beneath the list box.

NOTE:  The Plot button must be pressed after selecting pre-processing steps in order to generate an array of processed spectral data.

Plotting Spectra

To plot spectra, simply select one of the All or Single options, then click the plot button.
