PyChem (Python and Chemometrics) v2.0.0

A GUI for multivariate analysis which encompasses a number of methods including principal component analysis, cluster analysis, multivariate calibration, discriminant analysis as well as feature selection using genetic algorithm applications.

More information can be found at the project home page or at Sourceforge.  Please remember that PyChem is still under development, we're looking for people to contribute new functionality or enhance what is currently available, we would also be grateful for any feedback on operation of the software or bugs that you may uncover.  This can either be through email to  pychem-users@lists.sourceforge.net or the Sourceforge website.
