Partial Least Squares Regression

Partial least squares regression (PLSR) is used for formulating input to output transformation models based on supervised learning, which are predictive generally for quantitative traits.  In short, it allows the user to take multivariate inputs and generate a calibration model based on a single Y constituent (PLS1).  Although multiple constituents can be calibrated by repeated PLS1 runs.

Running PLSR

To run PLSR select either the Raw Data, or Processed data (if applicable) as the input for the algorithm.  The maximum number of PLS factors to be used can be altered by the user, although the algorithm will select the optimum number of factors based on the cross-validation error.  To run PLSR, press the Run PLS button.  For PLS, the dependent variables are described using a Class vector in the Experiment Setup screen

Outputs

(1) PLS calibration model, actual values (or Y-constituents) vs. the values predicted by the model

(2) PLS loadings for each latent variable extracted by the model

(3) The model predictions vs. the residuals.

(4) Error curves for the training and validation samples.  The optimal number of PLS factors (latent variables) used by the model correspond to the minimum error for the validation data set.

(5) Model errors for training, validation and independent test samples, plus the results for ordinary least squares analysis of the model.

Exporting Results

Pressing the Export button brings up a dialog which provides options for the export of PLSR results.
