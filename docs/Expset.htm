Experiment Setup

After loading an n x p 2D array through the ...File...Import menu option, two spreadsheet style grids are generated, which are selected using the  icon.  The Experiment Setup Parameters grid  should have n rows; here the labels used for plotting, the class structures applied for supervised learning algorithms, and the training, cross validation and test samples are defined.   Additional Label, Class or Validation columns can be added using the ,  and  icons respectively

The Independent variable labels grid should have p rows and can be used to enter vectors of text or numerical values that define the independent variables measured.  Values can be pasted in to these spreadsheet style grids or imported using a .csv file format, specified in the Getting Started section.

Adding New Columns

The columns in the grid are labelled automatically as Label, Class and Validation. To insert new columns simply press the appropriate button (defined above).  Randomly generated validation masks can also be created clicking on the Split Data checkbox.  Here, mask just refers to a vector of identifiers (which must be one of Train, Test or Validation) which tell the program how to split the data for the purpose of cross validation.  The proportion of samples to be split in to cross validation and independent test groups can also be defined.

Deleting Columns

To delete a column, select a cell in the column you want to delete, right click the mouse and select the delete option.

Renaming Columns

To rename a column, select a cell in the column you want to delete, right click the mouse and select the rename option.  Enter the new column heading in the dialog that appears and press enter.
