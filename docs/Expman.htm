Experiment Manager

Using the Experiment Manager, data can be saved in the PyChem format.  This allows for  analyses of each PyChem Experiment to easily be saved at stages along the process using the Snapshot functionality.

Use the Browse... button to select an empty folder where your work will be saved.

Experiments

Once a flat ASCII file with your 2D array of X-data has been loaded in to the application (see Getting Started), press the Create New Experiment button to save this for future use in PyChem.  A dialog box requesting a name under which to save the experiment appears.

Experiments can be renamed or deleted using the Rename Experiment and Delete Experiment buttons.

Snapshots

The idea of snapshots is to save the state of the system at a point in time.  So if some analysis has been performed that the user wishes retain, pressing the Capture Snapshot button after selecting an experiment from the tree will save all the current results and system settings.

Snapshots can be renamed or deleted using the Rename Snapshot and Delete Snapshot buttons.  To load a snapshot, select the snapshot you wish to load on the tree, and then press the Load Snapshot button.

Experiment and Snapshot Descriptions

Notes about experiments or snapshots can be saved using the experiment and snapshot description notepads.  To edit a description, select the experiment or snapshot on the tree, press the appropriate Edit button, type whatever it is you want to type and then press the Update button to save.
