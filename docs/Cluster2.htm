Cluster Analysis

Cluster analysis encompasses a number of unsupervised methods for grouping/separating similar/dissimilar objects.  PyChem provides a limited graphical interface to Michael Eisens' C-clustering library, which includes the K-means, K-medians, K-medoids and hierarchical clustering algorithms.

Running Cluster Analysis

Cluster analysis can be run on either raw or processed data, principal component or discriminant function scores.  The algorithm can be chosen by selecting the appropriate radio button in the Algorithms choice.  For any of the K-class algorithms, results are returned as a confusion matrix, whereas hierarchical clustering results are returned in the form of a tree diagram (or dendrogram).  The latter can be exported for visualisation using Alok Saldanha's freely available  Java TreeView program, which may be of more use for detailed analysis of these results.

Full details of the options available for running cluster analysis, such as explanations of the different distance functions can be found HERE.

Exporting results

Pressing the Export button brings up a dialog which provides options for the export of the dendrogram generated by hierarchical cluster analysis (HCA).  The format is suitable for viewing in the superior Java TreeView program by Alok Saldanha, which can be obtained from http://genome-www.stanford.edu/
