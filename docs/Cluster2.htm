Cluster Analysis

Cluster analysis encompasses a number of unsupervised methods for grouping/separating similar/dissimilar objects.  PyChem provides a limited graphical interface to Michael Eisens' C-clustering library, which includes the K-means, K-medians, K-medoids and hierarchical clustering algorithms.

Running Cluster Analysis

Cluster analysis can be run on either raw or processed data, principal component or discriminant function scores.  The algorithm can be chosen by selecting the appropriate option from the dropdown menu.  For any of the K-class algorithms, results are returned as a confusion matrix, whereas hierarchical clustering results are returned in the form of a tree diagram (or dendrogram).
