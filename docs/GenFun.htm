General Functionality
  1. Right clicking on any plot

 

Selecting Copy Figure will copy the plot to the clipboard so that it can be pasted as a bitmap in other Windows applications, such as MS Powerpoint.

Selecting Copy Coordinates will copy the XY coordinates to the clipboard so that the can be pasted in to other applications.

Selecting Print will open a print dialog so that the figure can be sent to a printer.

Selecting Save will open a dialog to allow the image to be saved to file.

Selecting Properties will open a plot properties dialog in order to change axis labelling, the title text, font sizes and axes ranges.
 

 

  1. Right clicking on the grid in Experiment Setup

Selecting Copy will copy the text to clipboard.

Selecting Paste will paste text in to grid cells.

Selecting Rename will invoke a dialog for changing column headings.  The options in the drop down selection menus will automatically be changed.

Selecting Delete Column will remove the currently active grid column.
