Discriminant Function Analysis

Discriminant function analysis (DFA), sometimes called canonical variates analysis (CVA), or Fisher discriminant analysis, is a supervised method for distinguishing between different sample groups and for subsequent predictions on new samples.  The algorithm uses the Fisher ratio with a priori knowledge of the class structure to maximise between group and minimise within group distances.

Running DFA

To run DFA select either the Raw Data, Processed data (if applicable), or Principal component scores as the input in to the algorithm.  Typically, for complex data where there are many hundreds of measured X-variables PC scores are used as input to DFA, this is to avoid singular matrices due to collinearities in the data, which create unacceptable conditions for this algorithm.  To run DFA, press the Run DFA button.

NOTE:  For arrays with large numbers of continuously measured x-variables, running DFA without performing an initial data reduction step is not advised.

Outputs

(1) DFA ordination plot, a 1D or 2D plot of the DFA scores output.

(2) DFA loadings 1D and 2D

(4) The eigenvalues (provide an approximation of the Fisher ratio)

(5) Plot of DF score mean group centres with 95% confidence circles for DF1 vs. DF2

Exporting Results

Pressing the Export button brings up a dialog which provides options for the export of DFA results.
